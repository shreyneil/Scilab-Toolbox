integralImage

Calculate the integral image of an input grayscale image

Calling Sequence

List output_image = integralImage(List input_image);
List output_image = integralImage(List input_image,method);

Arguments

input_image :

the input grayscale image matrix whose intrgralImage is to be calculated.

method :

method : 'upright' (default)

method : 'rotated' The area sums are calulated over a rectangle, which is rotated 45 degrees

Description

The function calculates the Integral Image of the input grayscale image.

Example

a = imread("gs.jpg");
b=integralImage(a);
imshow(b);


a = imread("gs.jpg");
b=integralImage(a,'rotated');
imshow(b);
