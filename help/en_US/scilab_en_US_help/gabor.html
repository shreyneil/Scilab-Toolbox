gabor

Generates a gabor kernel with the given wavelength and orientation

Calling Sequence

List kernel = gabor(double wavelength, double orientation);

Arguments

wavelength :

pixels/cycle of the sinusoidal carrier, must be >=2.

orientation :

orientation of the filter in degrees, must be between [0,360]

Description

The function generates a gabor kernel with the given wavelength and orientation.

Example

kernel=gabor(20,90);
imshow(kernel);
