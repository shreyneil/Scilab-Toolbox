imboxfilt

n*m box filter convolution in scilab with a padding type and normalization factor.

Calling Sequence

List output_image = imboxfilt(List Input_Image);
List output_image = imboxfilt(List Input_Image,kernel_size);
List output_image = imboxfilt(List Input_Image,[kernel_size_x,kernel_size_y]);
List output_image = imboxfilt(List Input_Image,kernel_size,padding_type);
List output_image = imboxfilt(List Input_Image,kernel_size,padding_type,Normalization_Factor);

Arguments

Input_Image :

the image matrix on which box filter has to be applied

kernel_size :

size of the box filter to be applied - must be an odd positive scalar >=3 - If not mentioned, default is 3

[kernel_size_x,kernel_size_y]:

size of the box filter applied, kernel_size can be replaced by a 1*2 vector of the kernel size in the x and y directions respectively

padding_type

value: 'default' : replicate the border padding, or take the value of the border pixel to be the closest image border pixel

value: 'symmetric' : take the border values to be a reflection of the image

value: 'circular' : take the border values assuming that the image is periodic

value: 'constant' : pad the image with the provided scalar double

NormalizationFactor :

value: any scalar: this is multiplied to the image after convolution

Description

The function smoothes an image using a n*m box filter.

Example

a = imread("lena.jpg");
b=imboxfilt(a);
imshow(b);


a = imread("lena.jpg");
b=imboxfilt(a,7);
imshow(b);


a = imread("lena.jpg");
b=imboxfilt(a,[9,11]);
imshow(b);


a = imread("lena.jpg");
b=imboxfilt(a,[9,11],'circular');
imshow(b);


a = imread("lena.jpg");
b=imboxfilt(a,[9,11],);
imshow(b);
