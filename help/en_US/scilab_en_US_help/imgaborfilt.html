imgaborfilt

Perform gabor filtering on a grayscale image with a gabor kernel of given wavelength and orientation

Calling Sequence

List output_image = gabor(List input_image,double wavelength, double orientation);

Arguments

input_image :

the input grayscale image matrix whose intrgralImage is to be calculated.

wavelength :

pixels/cycle of the sinusoidal carrier, must be >=2.

orientation :

orientation of the filter in degrees, must be between [0,360]

Description

The function performs gabor filtering on a grayscale image with a gabor kernel of given wavelength and orientation

Example

a = imread("gs.jpg");
b=imgaborfilt(a,2,0);
imshow(b);
